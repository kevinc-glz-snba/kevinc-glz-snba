### 👁️ Sistema de Identificación Facial en Tiempo Real para Medios
💼 Proyecto profesional / privado.

Sistema de reconocimiento facial orientado al análisis de señales de video en tiempo real.

Características principales:
- Arquitectura basada en microservicios.
- Backend desarrollado con FastAPI.
- Servidor independiente dedicado al procesamiento de IA.
- Reconocimiento facial utilizando InsightFace y FaceAnalysis.
- Procesamiento de video en vivo y contenido pregrabado.
- Dashboard web desarrollado con React y TypeScript.
- Gestión de información mediante PostgreSQL.

**Stack tecnológico:**
Python · FastAPI · OpenCV · InsightFace · React · TypeScript · PostgreSQL

### 📷 Sistema de Asistencia mediante Reconocimiento Facial
🎓 Proyecto académico.

Sistema automatizado para registrar asistencia mediante visión por computadora.

Características:
- Detección facial en tiempo real.
- Implementación con YOLO (Ultralytics).
- Generación automática de reportes.
- Optimización del proceso tradicional de pase de lista.

**Stack tecnológico:**
Python · YOLO · Computer Vision · OpenCV

### 🔐 Control de Acceso mediante NFC
🎓 Proyecto académico de integración hardware-software.

Sistema de autenticación mediante tecnología NFC para mejorar la seguridad de estaciones de trabajo.

Características:
- Diseño de circuito electrónico.
- Identificación mediante tarjetas NFC.
- Sistema automático de notificaciones de acceso.
- Integración entre hardware y software.

**Tecnologías utilizadas:**
NFC · Electrónica · Automatización · Programación embebida
